# Skill: reminder

## 描述
创建定时提醒，在指定时间向用户发送提醒消息。

## 触发条件
用户想要设置提醒、闹钟、定时通知，或要求在未来某个时间点被提醒做某事。

## 参数
- remind_time: 提醒时间（必需，支持自然语言如"明天上午9点"）
- message: 提醒内容（必需）
- repeat: 是否重复（可选，如"每天"、"每周一"）

## 约束
- 提醒依赖 delay/cron 调度基础设施
- 单次提醒通过 DelayStore 调度，重复提醒通过 CronStore 调度（重复提醒待 CronStore 就绪后启用）
- 提醒时间不能早于当前时间
- 提醒时间上限为 30 天（与 delay_seconds 上限一致）

## 示例对话
用户：明天早上9点提醒我准备开庭材料
