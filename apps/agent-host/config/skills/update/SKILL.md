# Skill: update

## 描述
更新多维表格中已有记录的字段值，支持单条和批量更新。

## 触发条件
用户想要修改、更新、变更多维表格中已有记录的某些字段内容。

## 参数
- table_name: 目标表格名称（必需）
- record_id: 要更新的记录 ID（必需，单条更新时）
- fields: 要更新的字段及新值（必需）
- filter: 批量更新时的筛选条件（可选，批量更新时使用）

## 约束
- 写链路需通过 locator triplet 校验（app_token + table_id + record_id 全部必填）
- 更新前需要用户确认（通过 PendingAction 确认流程）
- 不存在的字段名会被拒绝并返回错误提示
- 批量更新遵循 OperationEntry 逐条执行、失败即中止的策略

## 示例对话
用户：把案件编号 2024-001 的状态改成"已结案"
