# 飞书机器人模拟对话样例（基于当前多维表格）

以下样例基于当前可见案件数据（示例实体：`房怡康`、`深圳市神州红国际软装艺术有限公司`、案号 `（2024）粤01民终28497号`）整理，可直接在飞书私聊测试。

## 1. 案件查询（QuerySkill）

### 1.1 指定对象查询

- 用户：`帮我查一下深圳市神州红国际软装艺术有限公司的案子`
- 用户：`查询案号为（2024）粤01民终28497号的案件`
- 用户：`查看房怡康负责的案件`
- 用户：`帮我查一下深圳市荣创商贸有限公司的案子`

### 1.2 时间范围内开庭安排

- 用户：`这周有什么庭要开？`
- 用户：`明天上午有开庭吗？`
- 用户：`查询下个月的开庭安排`
- 用户：`2月20号有什么庭要开`

### 1.3 其他日期字段查询

支持根据语义识别日期字段并走区间查询：

- 用户：`这周有哪些管辖权异议截止的案件？`
- 用户：`查询下周举证截止的案件`
- 用户：`本月查封到期的案件有哪些？`
- 用户：`明天有上诉截止的案件吗？`

> 说明：当时间范围内无匹配记录时，开庭场景应回复“该时间范围内没有开庭安排”。

## 2. 提醒管理（ReminderSkill）

- 用户：`提醒我明天上午10点开会`
- 用户：`下周五下午3点提醒我提交报告`
- 用户：`10分钟后提醒我回个电话`
- 用户：`提醒我晚上8点去健身，优先级高`

- 用户：`我有哪些提醒？`
- 用户：`查看待办列表`
- 用户：`完成提醒 12`
- 用户：`删除提醒 #5`
- 用户：`取消提醒 3`

## 3. 闲聊与帮助（ChitchatSkill）

- 用户：`你好`
- 用户：`你是谁`
- 用户：`你能做什么？`
- 用户：`怎么使用？`

## 4. 组合意图（Planner & Router）

- 用户：`帮我查一下房怡康负责的案件，然后提醒我明天下午2点联系他`
- 用户：`把“准备开庭材料”加到我的待办里，这周三截止`

## 5. 预期行为（验收口径）

- 时间范围查询：有数据则返回记录，无数据返回“没有安排/没有匹配记录”提示。
- 人员查询：优先按人员字段检索（`主办律师`），可用姓名兜底匹配。
- 案号查询：应提取纯案号值，不应包含“案号为”“的案件”等噪声词。
