# OmniAgent 最终验收清单

## 1. 功能验收

### 1.1 飞书 Webhook
- [ ] URL 验证 (`url_verification`) 正常响应
- [ ] 消息加密解密正确
- [ ] 去重机制工作正常
- [ ] 仅处理私聊消息

### 1.2 意图识别
- [ ] 规则匹配：输入"查一下案件"能识别为 QuerySkill
- [ ] 链式意图：输入"帮我总结本周的庭"能触发 Query → Summary
- [ ] 兜底技能：无法识别时返回 ChitchatSkill

### 1.3 技能执行
- [ ] QuerySkill：查询案件返回正确结果
- [ ] SummarySkill：汇总结果格式正确
- [ ] ReminderSkill：创建提醒成功
- [ ] ChitchatSkill：问候/帮助响应正确

### 1.4 消息回复
- [ ] 文本消息格式正确
- [ ] 卡片消息渲染正确
- [ ] 回复关联原消息

## 2. 性能验收

### 2.1 响应时间
- [ ] 简单查询 < 2s

### 2.2 资源占用
- [ ] 内存 < 500MB（正常负载）
- [ ] CPU 稳定无持续高占用

### 2.3 并发处理
- [ ] 多用户同时请求不阻塞
- [ ] 无资源泄露
